# Changelog

All notable changes to the Instamine Mod will be documented in this file.

## [1.0] - 2025-11-29

### Added
- Initial release of Instamine Mod
- Instant mining for all tools (999,999 tool DPS)
- One-shot block breaking (999,999 hit damage)
- Near-instant attack animations (1ms minimum)
- Zero cooldown between mining actions
- Massive combat damage boost (1,000,000 damage)
- Support for all tool types (pickaxes, axes, shovels)
- Support for all weapon types
- Client-side only implementation (no servermod required)
